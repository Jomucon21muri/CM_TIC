# Tipos de Licencias de Software

## 1. Software Propietario (Copyright)

!!! warning "Características del Software Propietario"
    - El código fuente no está disponible
    - No se puede modificar ni redistribuir
    - Requiere licencia de pago o aceptación de términos
    
    **Tipos:**
    
    - **Comercial:** Se paga por usarlo (Microsoft Office, Adobe Photoshop)
    - **Freeware:** Gratuito pero propietario (Skype, Adobe Reader)
    - **Shareware:** Prueba gratuita limitada, luego de pago (WinRAR)
    - **Demo:** Versión de demostración con funciones limitadas
    - **Trial:** Versión completa con tiempo de uso limitado
    
    **Ejemplos:** Windows, Microsoft Office, Adobe Creative Suite

## 2. Software Libre y Código Abierto

!!! success "Software Libre"
    ### GPL (General Public License)
    
    - Creada por la Free Software Foundation
    - **4 libertades fundamentales:**
        - **Libertad 0:** Usar el programa para cualquier propósito
        - **Libertad 1:** Estudiar cómo funciona y modificarlo
        - **Libertad 2:** Redistribuir copias
        - **Libertad 3:** Distribuir versiones modificadas
    - **Copyleft:** Las obras derivadas deben usar la misma licencia
    - **Ejemplo:** Linux, GNU
    
    ### Licencias BSD, MIT, Apache
    
    - Más permisivas que GPL
    - Permiten uso en software propietario
    - No requieren copyleft
    
    ### Open Source
    
    - Código fuente disponible
    - Se puede modificar y redistribuir
    - **Ejemplos:** LibreOffice, GIMP, Firefox, Linux, Android

## 3. Creative Commons (CC)

!!! info "Licencias Creative Commons"
    Licencias para obras creativas (no solo software): textos, imágenes, música, vídeos.
    
    ### Elementos de las Licencias CC:
    
    - **BY (Atribución):** Debe reconocerse la autoría
    - **SA (Share Alike):** Compartir igual, obras derivadas con misma licencia
    - **ND (No Derivatives):** No se permiten obras derivadas
    - **NC (Non Commercial):** Solo uso no comercial
    
    ### Combinaciones Comunes:
    
    - **CC BY:** Se puede usar libremente con atribución
    - **CC BY-SA:** Igual que BY, obras derivadas con misma licencia
    - **CC BY-NC:** Uso no comercial con atribución
    - **CC BY-NC-SA:** No comercial, atribución, compartir igual
    - **CC BY-ND:** Atribución, sin obras derivadas
    - **CC BY-NC-ND:** No comercial, atribución, sin derivadas (más restrictiva)
    - **CC0:** Dominio público, sin restricciones
    
    ### Ejemplos de Uso:
    
    - Wikipedia: CC BY-SA
    - Flickr, YouTube: Varias opciones CC
    - Recursos educativos abiertos (OER)

## Comparativa de Licencias

| Tipo | Código Fuente | Modificable | Redistribuir | Uso Comercial | Ejemplos |
|------|---------------|-------------|--------------|---------------|----------|
| **Comercial** | No disponible | No | No | Requiere pago | MS Office, Photoshop |
| **Freeware** | No disponible | No | Limitado | Gratis | Adobe Reader, Skype |
| **Shareware** | No disponible | No | No | Prueba/Pago | WinRAR |
| **GPL** | Disponible | Sí | Sí con GPL | Sí | Linux, GNU |
| **MIT/BSD** | Disponible | Sí | Sí cualquier licencia | Sí | jQuery, Node.js |
| **CC BY** | N/A | Sí | Sí con atribución | Sí | Muchas imágenes online |
| **CC BY-NC** | N/A | Sí | Sí con atribución | No | Recursos educativos |
