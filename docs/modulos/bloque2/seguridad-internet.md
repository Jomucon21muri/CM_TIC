# 2.3. Seguridad en Internet

## Técnicas de Fraude

### Phishing

!!! danger "¿Qué es el Phishing?"
    El **phishing** es un intento de obtener información confidencial (contraseñas, datos bancarios) haciéndose pasar por una entidad de confianza.
    
    #### Características del Phishing:
    
    - Correos que imitan a bancos, PayPal, Amazon, etc.
    - Urgencia ("tu cuenta será bloqueada")
    - Enlaces a páginas web falsas muy similares a las reales
    - Solicitan datos personales o bancarios
    - Errores ortográficos y gramaticales
    
    #### Cómo Protegerse:
    
    - Verificar remitente del correo
    - No hacer clic en enlaces sospechosos
    - Comprobar URL del sitio web (https://, dominio correcto)
    - Nunca proporcionar contraseñas por correo
    - Contactar directamente con la empresa ante dudas

### Pharming

- Redirección fraudulenta a sitios web falsos
- Manipula el DNS para enviar al usuario a páginas falsas
- Más sofisticado que el phishing

### Smishing

- Phishing por SMS
- Mensajes con enlaces maliciosos
- Ejemplo: "Has ganado un premio, haz clic aquí"

### Vishing

- Phishing por llamada telefónica
- Se hacen pasar por soporte técnico u organizaciones

### Ingeniería Social

- Manipulación psicológica para obtener información
- Suplantación de identidad
- Explotan la confianza humana

### Ataques Man-in-the-Middle (MITM)

- Interceptación de comunicaciones entre dos partes
- El atacante se sitúa entre usuario y servicio
- Puede leer o modificar la información

## Seguridad en el Correo Electrónico

!!! warning "Buenas Prácticas"
    - No abrir archivos adjuntos de remitentes desconocidos
    - Verificar extensiones de archivos (.exe, .bat son peligrosos)
    - Usar filtros antispam
    - No responder a correos sospechosos
    - Usar contraseñas fuertes y únicas
    - Activar autenticación de dos factores (2FA)
