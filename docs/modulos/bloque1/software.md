# 1.3. Software: Concepto y Clases

## Concepto

El **software** es el conjunto de programas, instrucciones y datos que permiten al hardware realizar tareas específicas.

## Clasificación del Software

### Software de Sistema

- **Sistemas Operativos:** Windows, Linux, macOS, Android, iOS
- **Controladores (Drivers):** Software que permite al SO comunicarse con el hardware
- **Utilidades:** Programas de mantenimiento, antivirus, compresores

### Software de Aplicación

- **Ofimática:** Procesadores de texto, hojas de cálculo, presentaciones
- **Navegadores web:** Chrome, Firefox, Edge
- **Multimedia:** Reproductores de audio/vídeo, editores de imagen
- **Comunicación:** Clientes de correo, mensajería instantánea

### Software de Programación

- **Compiladores e intérpretes:** Traducen código fuente a lenguaje máquina
- **Entornos de desarrollo (IDE):** Visual Studio, Eclipse, PyCharm
- **Depuradores:** Herramientas para encontrar errores en el código
