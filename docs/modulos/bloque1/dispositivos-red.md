# 1.7. Dispositivos de Red

## Tarjeta de Red (NIC)

- Permite conectar un ordenador a la red
- Puede ser ethernet (RJ-45) o inalámbrica (Wi-Fi)
- Tiene una dirección MAC única

## Hub (Concentrador)

- Dispositivo básico que conecta varios equipos
- Envía datos a todos los puertos (no inteligente)
- Actualmente en desuso

## Switch (Conmutador)

- Conecta varios dispositivos en una LAN
- Envía datos solo al dispositivo destinatario (inteligente)
- Trabaja en la capa 2 (enlace de datos) del modelo OSI

## Router (Enrutador)

- Conecta diferentes redes entre sí
- Dirige el tráfico de datos entre redes
- Asigna direcciones IP (con DHCP)
- Proporciona seguridad (firewall)
- Ejemplo: router de casa que conecta tu LAN con Internet

## Punto de Acceso (Access Point)

- Crea una red Wi-Fi
- Conecta dispositivos inalámbricos a una red cableada

## Módem

- Modula y demodula señales
- Convierte señales digitales a analógicas y viceversa
- Necesario para conexiones telefónicas, cable o fibra óptica

## Repetidor

- Amplifica la señal para extender el alcance de la red
- No modifica la información

## Bridge (Puente)

- Conecta dos segmentos de red
- Filtra tráfico entre segmentos
