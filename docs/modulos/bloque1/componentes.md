# 1.2. Componentes Físicos del Ordenador Personal

## Hardware

El **hardware** es el conjunto de componentes físicos y tangibles de un ordenador.

## Componentes Principales

### Unidad Central de Proceso (CPU)

- **Microprocesador:** "Cerebro" del ordenador que ejecuta instrucciones
- **Velocidad:** Medida en GHz (Gigahercios)
- **Núcleos:** Procesadores individuales en un chip (dual-core, quad-core, etc.)
- **Fabricantes principales:** Intel, AMD

### Memoria Principal

- **RAM (Random Access Memory):** Memoria volátil de acceso aleatorio
    - Almacena temporalmente datos y programas en ejecución
    - Se borra al apagar el ordenador
    - Mayor RAM = mejor rendimiento multitarea

- **ROM (Read Only Memory):** Memoria de solo lectura
    - Contiene instrucciones permanentes (BIOS/UEFI)
    - No se borra al apagar el ordenador

### Placa Base (Motherboard)

- Circuito impreso principal que conecta todos los componentes
- Contiene el chipset que coordina el flujo de datos
- Incluye ranuras de expansión (PCI, PCIe)
- Conectores para CPU, RAM, almacenamiento y periféricos

### Dispositivos de Almacenamiento

- **Disco Duro (HDD):** Almacenamiento magnético, más capacidad, más lento
- **Disco de Estado Sólido (SSD):** Sin partes móviles, más rápido, más caro
- **Unidades ópticas:** CD, DVD, Blu-ray
- **Almacenamiento externo:** USB, tarjetas SD, discos externos

### Tarjeta Gráfica (GPU)

- Procesa y genera imágenes para el monitor
- Integrada en la CPU o dedicada (tarjeta independiente)
- Esencial para videojuegos, edición de vídeo, diseño 3D

### Fuente de Alimentación

- Convierte corriente alterna (AC) a corriente continua (DC)
- Suministra energía a todos los componentes
- Medida en vatios (W)

### Periféricos de Entrada

- Teclado, ratón, touchpad
- Escáner, webcam, micrófono
- Pantalla táctil, lector de códigos de barras

### Periféricos de Salida

- Monitor, proyector
- Impresora, plotter
- Altavoces, auriculares

### Periféricos de Entrada/Salida

- Pantalla táctil
- Impresora multifunción (imprime y escanea)
- Cascos de realidad virtual
