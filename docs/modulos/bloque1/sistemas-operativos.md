# 1.4. Sistemas Operativos: Definición y Funciones

## Definición

Un **Sistema Operativo (SO)** es el software fundamental que gestiona el hardware y proporciona servicios a los programas de aplicación. Actúa como intermediario entre el usuario y el hardware.

## Funciones Principales

- **Gestión del procesador:** Administra el tiempo de CPU entre procesos
- **Gestión de memoria:** Asigna y libera memoria RAM
- **Gestión de entrada/salida:** Controla periféricos y dispositivos
- **Gestión de archivos:** Organiza y accede a datos en almacenamiento
- **Interfaz de usuario:** Permite la interacción (GUI o CLI)
- **Seguridad:** Control de acceso y protección de datos

## Ejemplos de Sistemas Operativos

### Para Ordenadores de Escritorio

- **Windows:** SO más utilizado, de Microsoft (Windows 10, 11)
- **macOS:** SO de Apple para ordenadores Mac
- **Linux:** SO de código abierto (Ubuntu, Fedora, Debian)

### Para Dispositivos Móviles

- **Android:** Basado en Linux, de Google
- **iOS:** SO de Apple para iPhone y iPad
- **HarmonyOS:** SO de Huawei

### Para Servidores

- **Windows Server:** Versión de Windows para servidores
- **Linux:** Muy utilizado en servidores (Ubuntu Server, CentOS)
- **Unix:** Sistemas tradicionalmente usados en servidores grandes
