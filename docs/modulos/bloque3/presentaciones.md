# 3.4. Presentaciones

Software para crear presentaciones visuales con diapositivas.

**Ejemplos:** Microsoft PowerPoint, LibreOffice Impress, Google Slides

## Elementos de una Presentación

- **Diapositiva:** Cada pantalla individual de la presentación
- **Diseño:** Plantilla de distribución de elementos en la diapositiva
- **Tema:** Conjunto de colores, fuentes y efectos
- **Patrón:** Diseño maestro que aplica a todas las diapositivas

## Creación de Diapositivas

- Nueva diapositiva: Insertar diapositiva con diseño predefinido
- Tipos de diseños: Título, contenido, dos columnas, imagen con título, etc.
- Orden: Se pueden reorganizar arrastrando

## Inserción de Elementos Multimedia

### Texto

- Cuadros de texto
- WordArt (texto decorativo)
- Formato: fuentes, tamaños, colores, efectos

### Imágenes

- Desde archivo local
- Imágenes en línea
- Capturas de pantalla
- Recorte, compresión, efectos artísticos

### Formas y SmartArt

- Formas geométricas, flechas, líneas
- SmartArt: Diagramas profesionales (organigramas, procesos, ciclos)

### Audio y Vídeo

- Insertar archivos de audio (música, narración)
- Insertar vídeos (archivo o enlace de YouTube)
- Configurar reproducción automática o al hacer clic

### Tablas y Gráficos

- Tablas para organizar información
- Gráficos para representar datos

### Hipervínculos

- Enlaces a páginas web
- Enlaces a otras diapositivas
- Enlaces a archivos externos

## Efectos y Animaciones

### Transiciones

- Efectos al pasar de una diapositiva a otra
- Tipos: Desvanecer, empujar, barrido, cortinilla, etc.
- Duración y sonido configurables

### Animaciones

- Efectos sobre elementos dentro de una diapositiva
- **Entrada:** Aparecer, desvanecerse, volar desde
- **Énfasis:** Girar, cambiar tamaño, parpadear
- **Salida:** Desaparecer, volar hacia
- **Trayectorias de movimiento:** Personalizar recorrido

### Buenas Prácticas

!!! warning "Uso de animaciones"
    - Usar animaciones con moderación (no abusar)
    - Coherencia en el estilo de transiciones
    - Velocidad adecuada (ni muy lenta ni muy rápida)

## Consejos para Presentaciones Efectivas

!!! success "Recomendaciones"
    - **Menos es más:** No saturar con texto
    - **Regla 6x6:** Máximo 6 líneas por diapositiva, 6 palabras por línea
    - **Contraste:** Texto legible sobre fondo
    - **Imágenes de calidad:** Alta resolución
    - **Consistencia:** Mismo estilo en toda la presentación
    - **Tamaño de fuente:** Mínimo 24 puntos para ser legible
