# 3.2. Procesadores de Texto

Aplicaciones para crear, editar y dar formato a documentos de texto.

**Ejemplos:** Microsoft Word, LibreOffice Writer, Google Docs

## Funciones Principales

### Edición de Texto

- Escribir, copiar, pegar, cortar texto
- Buscar y reemplazar
- Corrección ortográfica y gramatical
- Sinónimos y diccionarios

### Formato de Texto (Fuentes)

- **Tipo de fuente:** Arial, Times New Roman, Calibri, etc.
- **Tamaño:** Medido en puntos (pt)
- **Estilo:** Negrita, cursiva, subrayado, tachado
- **Color:** Del texto y resaltado
- **Efectos:** Superíndice, subíndice, mayúsculas/minúsculas

### Formato de Párrafo

- **Alineación:** Izquierda, derecha, centrada, justificada
- **Interlineado:** Espacio entre líneas
- **Espaciado:** Antes y después del párrafo
- **Sangría:** Primera línea, izquierda, derecha, francesa
- **Listas:** Numeradas, viñetas, multinivel

### Tabulaciones

- Alinear texto en posiciones específicas
- Tipos: Izquierda, derecha, centrada, decimal
- Relleno: Puntos, guiones, líneas

### Estilos y Plantillas

- **Estilos:** Conjunto de formatos predefinidos (Título 1, Título 2, Normal)
- **Plantillas:** Documentos con formato y estructura predefinida
- **Ventajas:** Consistencia, rapidez, fácil actualización

### Inserción de Elementos

- **Imágenes:** Desde archivo, online, capturas
    - Ajuste de texto: en línea, cuadrado, estrecho, detrás, delante
    - Recorte, rotación, efectos
- **Tablas:** Filas y columnas para organizar datos
- **Formas:** Rectángulos, círculos, flechas, etc.
- **Gráficos:** SmartArt, organigramas
- **Símbolos y ecuaciones matemáticas**
- **Hipervínculos:** Enlaces a páginas web o partes del documento

### Tablas de Contenido e Índices

- **Tabla de contenido:** Lista automática de capítulos basada en estilos de título
- **Índice alfabético:** Lista de términos con números de página
- **Referencias cruzadas:** Enlaces internos del documento

### Encabezados y Pies de Página

- Texto que aparece en la parte superior (encabezado) o inferior (pie) de cada página
- Pueden incluir: numeración de páginas, título del documento, fecha, autor
- Pueden ser diferentes en páginas pares/impares o primera página

### Maquetación

- **Márgenes:** Superior, inferior, izquierdo, derecho
- **Orientación:** Vertical (portrait) u horizontal (landscape)
- **Tamaño de papel:** A4, Carta, Legal, etc.
- **Columnas:** Diseño en varias columnas (estilo periódico)
- **Saltos:** De página, de sección, de columna

### Conversión a PDF

- **PDF** (Portable Document Format): Formato universal para documentos
- **Ventajas:** Aspecto idéntico en cualquier dispositivo, no editable fácilmente
- **Exportar/Guardar como PDF:** Función en todos los procesadores modernos
